# サイト概要

このサイトは学術系コンテンツを Markdown/MDX で運用し、静的配信に最適化することを目的としています。

## 主要機能

- MD/MDX のみでページを追加・更新
- タグとカテゴリで一覧・詳細ページを構成
- 「入門/専門」切り替えを同一URL内で提供
- MDX 内でインタラクティブグラフを埋め込み

## 運用ルール

- `created` は初回作成時のみ設定
- `updated` は内容変更時に自動更新
- `author` は手動または自動挿入
- 運用の手順は `usage.md` を参照
