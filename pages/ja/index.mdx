# 学習アウトプットサイト

数理生物学を中心に、学術系コンテンツを Markdown/MDX で継続的に更新するためのデモサイトです。

- 入口: `overview` にサイト方針と機能要約があります。
- 実例: `biology/lotka-volterra` に入門/専門の切り替えとインタラクティブグラフがあります。
- 収集: `tags` と `categories` に一覧と詳細サンプルがあります。

## 言語切り替え

- 日本語: `/ja`
- English: `/en`
