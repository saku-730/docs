# Site Overview

This site publishes academic content in Markdown/MDX and targets static hosting.

## Core Features

- Add and update pages with MD/MDX only
- Tag and category listing pages
- Beginner/Expert switch inside the same URL
- Interactive charts embedded in MDX

## Operations

- `created` is set once on creation
- `updated` is refreshed on content changes
- `author` can be manual or auto
