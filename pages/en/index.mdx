# Learning Output Site

This is a demo site for publishing academic notes in Markdown/MDX, focused on mathematical biology.

- Start: `overview` explains the intent and features.
- Example: `biology/lotka-volterra` shows beginner/expert tabs and an interactive chart.
- Browse: `tags` and `categories` provide listing samples.
